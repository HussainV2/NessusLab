<h1>Vulnerability Management With Nessus</h1>



<h2>Description</h2>
Project covers vulnerabilty scanning and vulnerability remediation. I used Nessus Essentials to scan local virtual machines hosted on VMware in order to run credentialed scans to discover vulnerabilities, remediate vulnerabilities, then perform a rescan to verify remediation.
<br />



<h2>Utilities Used</h2>

- <b>Nessus Essential, VMware</b>

<h2>Environments Used </h2>

- <b>Windows</b>
